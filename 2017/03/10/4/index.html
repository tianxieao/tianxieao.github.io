考友经验之综合篇3

发表于 2017-03-10 | 分类于 QZZN | | 阅读次数

转自从120分到150分的经历与感悟

作者：知行阁主

公考是个选拔性考试，无论是国考还是省考，竞争激烈程度平均下来至少在20比1以上。也就是说，要想考上普通岗位的公务员，难度相当于在一个60人的班级里考到前3名。大家可以回忆一下，从幼儿园算起，自己成绩排在班级前3名的次数有多少？实事求是地说，对于大多数参加公考的同学来说，公考一点也不容易，要不怎么会有人花十几万元报个包过的天价班。虽说近年公考有所降温，但随着经济下行，企业转型升级，就业的形势也日益严峻，目前及未来几年，考公务员还是很多人的选择。

我的公考之路像大多数同学一样，充满艰辛，硬是靠着百折不挠、屡败屡战的精神，最终于大学毕业2年后，也就是2013年在第6次考试以129分考取某县县委一核心部门。在机关工作两年后，2015年下半年开始，又参加一次省直事业单位考试（笔试第二，200多人竞争），参加两次省考（均以岗位第一名进入面试），最终以笔试152分（行测74，申论78），面试85分，考取某省会城市一职位，成为很多人眼中的高手。

回想2011年春季第一次参加选调生考试以120分的成绩败北，以及之后一系列丧家之犬般的奔波与失望，看着书房里堆积如山的书籍（各种版本的教材、专项训练、试卷，以及自己不计代价花重金、精力搜集来的视频、秘笈），感触良多。客观地讲，它们当中真正起作用的很少，甚至连十分之一都不到，有的甚至会起到误导作用，直接让我在某次面试误入歧途、前功尽弃。例如，一本面试书中这样说到:“在面试中，思考时间最好不要超过三十秒，这样会显得果断，给考官更优秀的感觉”，但事实上，由于面试材料很长，30秒根本没来得及看完，要点也没有提取出来，所以第一道题综合分析，由于没思考充分，就卡壳了，考场气氛很是尴尬，直接影响到后续题目的发挥，最终以全场倒数第二的成绩败给笔试的第二名与第三名。今年夏天我才知道，公考面试首要的是流畅，为保证流畅，可以思考1分钟到1分半钟，都是正常范围，这只是我印象比较深刻的一个例子。然而，要想在浩如烟海的公考资料中分辨出最有价值的，真的很难，这也正是大多数人多年考不上、觉得公考很难的原因。

其实，要想考上公务员，也并非很多人想象的那样困难，只要有决心，有高人指导，有精品的资料，普通人通过三个月到半年的准备就能轻松考取30人以下竞争的岗位。通过半年努力，就能冲击100人左右竞争力的岗位，而天资聪颖、善于学习的考生花费的时间会更少。想到我自己，由于最初意志不坚定，缺乏指导，去全国各地考试的花费，在社会上打工吃的苦头及损失的机会成本，可以说是十分巨大的，数量关系比较好的同学可以算上一算。所以，能早上岸一年就不要晚一年，否则整个命运都会发生改变，甚至会像我一样付出沉重的代价。

说到公考的辛酸，每个人多少都会有所体会，不再详述，只知道我在漫长的公考期间，女朋友分手了，头发一夜之间白了三根，换了三份工作，考了四个省份，去了六个城市，算上之前已成功的，到现在已考了十次。有人说，失败一定有原因，成功一定有方法，的确如此。我的成绩提升也有着明显的阶段性，而且一次比一次高一个台阶。原因很简单，要么是我认识了一个朋友，通过我请客吃饭他给了我一份珍藏的秘笈，要么是我通过购买资料，从中发现了一份真正有意义的宝典，要么是我遭受了重大的打击，开始下决心改变思维模式与学习习惯。例如，2013年在大学图书馆复习时认识的一个考友给的一份资料让我的言语理解的正确率从60%迅速提升到90%，直接促使我的行测从60分出头跨越到最高时78分（之前的一次考试），当然也与我悟出来行测要学会舍得，要练成本能的缘故；2015年购买的一份申论资料，让我突破了对申论答题的认知，使我的申论从五、六十分提升到最高时78分（最近的一次考试）；对命运的感悟，也让我变得谦虚，感恩，自律，求知若渴，因此也遇到很多高人的指点与鼓励，其中一位告诉我一句最励志的话：“你一定能考上公务员的，凡是我身边坚持考的，最后都考上了”，至今我仍记得那个人说话时的神情，那是一个县级检察院的公务员，是我在学校图书馆遇到的另外一个考友。

公务员工作虽然不错，但也不像很多人幻想的那么美好，也并非每个人都适合，在考之前，必须仔细考虑清楚。坚决要考的，就要尽最大努力，在最短的时间内考取理想的岗位，避免像我一样走弯路。如果您能看到这篇文章，说明你是幸运的，至少你可以思考你现在正在走的复习之路是否是弯路，你身边是否也有同在考试可以给你提供好资料的贵人。尤其是申论，更主要的是一种思维，而不是一味地闷着头学习，不管方向正确与否。
当然，公考也是一个相对漫长的征途，必须有足够的精神力量支撑自己。现在，再送上本人通往心灵自由的心路历程以及给我注入无畏勇气与力量的几本好书，还有曾经在本论坛取得的一份选词填空词库（已整理成省纸打印版，方便大家学习）。希望能帮助更多处在迷茫之中的考生攻坚克难，早日找到正确的生活道路，开启五彩缤纷的人生。

个人真实经历，希望能让大家有所启发，不能有所启发也请大家保持克制，不要随便扣帽子定性。送给尚未考上公务员的同学一句话，这句话支撑了我五年：“凡是我身边坚持考公务员的，最后都考上了”。言语秘笈深受好评，但部分资料涉及版权，上传会引发不必要麻烦，想要的同学请直接发消息。若有空会一一回复。
