考友经验之综合篇2

发表于 2017-03-09 | 分类于 QZZN | | 阅读次数

转自安哥说公考刻意练习法

作者：安哥分享

第一篇 备考经历

一直想着要回馈网络，毕竟我两次公考两次上岸都要感谢网络上很多前辈的无私分享，当然也很感谢实际指导我面试的某个人社局老前辈，曾经的我也在备考的辛苦中激励着自己“嘿嘿，等我哪一天上岸了，也像大牛们一样来这里分享经验，想想就觉得很爽呢！”如今已经到了2017年，上班也有一段时间了，回顾2016，谨以此贴纪念考公的日子吧！
一、个人情况
我大学毕业于传说中全国最美的大学之一，具体是哪个？你们自己猜…毕业后，因为年轻不懂事，也去了北上广深某一城市，在一个著名民企做财务，收入呢，和该地的房价严重不对等，再加上独生子女的缘故，父母家人常常打电话劝我回来，所以就决定考个公务员试试看吧，所以2015年的国考，就报了家乡省会城市国税的岗位，招一个，我笔试第三进面，后来成功翻盘，总分第一。2016年初，省考又开始了，本来我是拒绝的，毕竟可是duang,看到一个非常心动的岗位，传说中的m委，据说工作清闲有前途，好吧，那就去试试，后来结合我之前备考的经验，最后笔试、面试双第一，成功上岸，当然就选择去*委上班了。

二、上岸经验

说我的考公经历之前，我想和大家说一本最近大热的书《刻意练习》，这是决定你的备考方法是否有效的关键，请大家一定要重视，一定要重视，一定要重视！先来一段豆瓣的简介：著名心理学家艾利克森在“专业特长科学”领域潜心几十年，研究了一系列行业或领域中的专家级人物：国际象棋大师、顶尖小提琴家、运动明星、记忆高手、拼字冠军、杰出医生等。

他发现，不论在什么行业或领域，提高技能与能力的最有效方法全都遵循一系列普遍原则，他将这种通用方法命名为“刻意练习”。对于在任何行业或领域中希望提升自己的每个人，刻意练习是黄金标准，是迄今为止发现的最强大的学习方法。所有人都以为“杰出”源于“天赋”，“天才”却说：我的成就源于“正确的练习”！这本书其实是我最近才看到的，但是看了之后我突然醍醐灌顶，为什么考公的时候，别人总说我很幸运，一考就考上2个，而身边的同学有的考了几次都没上岸，我曾经以为是运气的成分，亦或者是我比别人勤奋些，可是仔细想想，真的是这样吗？我国考和省考的时候，竞争对手既有大学生村官，也有事业单位在职人员，而我之前在企业上班，按说在申论和面试上和他们比都没有优势，毕竟对体制内的很多做事程序都不了解啊！现在结合这本书的理论，我意识到了，我自己不论是在笔试、还是面试准备的时候，都无意的做到了刻意练习，而其他人，无论他们付出了多少时间和精力，交了多少培训费，如果没有达到刻意练习的境界，那么这样的努力只是自欺欺人，对你的实际笔试和面试水平并不会有任何帮助，所以，在我们身边，总会听到有考友说：我考了几年，每次都只差几分！很多人总以为这几分可以通过多考一年、多复习一年来弥补，事实上，如果你的方法不对，这个差距是无法弥补的。好了，说到这里，有人肯定会马上去搜《刻意练习》这本书，那我在这里把书中的理论一句话概括下吧：“坚持去练习你觉得困难的事情”，结合到我们的考公就是“坚持去做你觉得难的题、坚持去答你觉得难的题目！”当然，在刻意练习的过程中也要接受正确及时的反馈，以对自己的练习进行不断调整，最后成为高手！

考友经验之申论篇1

发表于 2017-03-08 | 分类于 QZZN | | 阅读次数

转自16年行测68+，申论74+考友经验篇

作者：东东ricky

楼主本人参加了今年的北方某省联考，行测68+，申论分数74+，现已成功上岸。今年备考期间在论坛收获颇多，当时有一些个人觉得不错的学习经验，以一篇干货贴的形式发在申论版块回报论坛，（《一篇一万字的干货—拨云见日，以16年国考真题深论概括题和综合分析题（更新版）》），原帖地址：http://bbs.qzzn.com/thread-15268668-1-1.html 获得不少考友的肯定，阅读80000+，深感荣幸，推崇备至，愧不敢当。很遗憾，此贴现已看不了，当时初入论坛也不懂得什么规矩，在文章结尾留下过qq号以便和大家交流，后来被版主以违规为由屏蔽了。有的朋友还在帖子里问能不能找到，在这里我会贴出一部分精华，供考友们了解申论解题之道，破除申论迷雾。此外，一些我个人觉得学习申论的几个误区和重要的学习方法，也会跟考友们分享分享。此为论坛封笔贴，愿诸君成功上岸，达成梦想。从此飘然远去，高山流水，扁舟一叶横流，闲看庭前花开花落，漫随天外云卷云舒。

一、申论考察的实质

申论作为一门以考察考生文字表达能力和阅读理解能力为主的考试科目，在被妖魔化多年后，已然被广大考生视为洪水猛兽。确实，传统的中国式教育使我们绝大数考生已失去了独立分析思考的能力，同时严重依赖固有标准，离开了标准，就会陷入盲目迷茫无助的困惑中。—-申论没有标准答案要如何写？这个问题让许多考生只能盲目把希望寄托于某些知名培训机构或网络上的风言风语，从而失去该有的判断力—作为一个阅读者，我如何理解给定材料内容？作为一个问题的回答者，我要如何全面准确的表达出我的思想和内容？
因此，真正的答案永远不会公布在你的面前，只能抓住思考的真理—以给定材料为内容基准，以逻辑理论为答案结构，以语言风格为应试技巧。从内容的全面性准确性，结构的逻辑性和条理性，语言上的简练性和针对性这三个出发点确立自己的答案，使判卷人心悦诚服。道理说起来简单做起来难，这不仅要求诸位了解这种正确的申论思维，更要自己的勤奋努力，持之以恒的练习，纠正，从而突破固有思维进化。

二、申论的误区—以模板行天下

申论反模板化的趋势已经越来越明显，许多考生也注意到了这个问题，但还是会在学习的过程中不自觉的使用模板思维，这不仅仅体现在大作文中，更是体现在小题中。模板化这种行为从根本上脱离了申论考试考察的意义，更容易使很多考生犯下脱离材料这种严重的硬伤，因此，模板是大忌，也是大家想要突破重围杀向成功彼岸的拦路虎，以模板的思维想要在申论考试中拿高分，简直比中六合彩都难。

举个例子吧，有个考友曾经问我，楼主做小题的时候如何提高概括能力呢？像通俗化，标准化，延展性这类的词总是在嘴边，就是做题的时候想不起来，看着划好的材料句子也没用。我给他的回答是：这些词根本在考试的时候用不到，需要概括的是材料核心，而非硬套这些本无关的模板词汇。

此外，还有很多同学依旧喜欢“加大宣传力度，完善法律法规”这类的空洞式语言作为答案，此类答案体现不出你个人一丁点的能力，这样的问题也许就是你申论低分所在。所以，学习申论最重要的就是掌握正确的申论实战思维，而不是背模板，这样才能真正做到以不变应万变，在考试中游刃有余。

考友经验之综合篇1

发表于 2017-03-07 | 分类于 QZZN | | 阅读次数

转自16年国考148分考友经验篇

作者：扑腾的丑小鸭

开始公考的时候在论坛下了很多资料，各路大神的帖子资料都避免了我走很多弯路，现在进入政审，为以后的公考人提供自己的一点经验吧，希望能够对大家有所帮助。

先给大家说一下我的考试历程：2015年国考裸考90.5分，基本线都没有达；2015年省考120分，当时在学校来说算是高分，但是没进面试。不过准备考试可以说是从这个时候开始的；然后2016年国考，148进面。所以，无论你现在考试处于什么水平，好好准备就一定会有提高的。

我的备考大致可以分为以下两个阶段。

一 懵懂阶段

15年省考的时候开始准备考试，刚开始完全不知道该学些什么，而且由于个人比较懒，报名之后省考才开始准备，时间非常紧张，买书来看就有点来不及了。因此，开始准备的时候直接用的真题卷。如果你也是刚开始准备公考对自已是什么水平没有把握，我建议你也开始就从真题做起。大概做个5，6份卷子，统计一下自己每个部分的正确率，所用时间，应该就能对自己的弱项强项有个大概的把握。接下来就可以有针对性的进行突破了。

给大家说一下我当时的情况，言语部分一般错误在10个左右，逻辑8个，用时一般在27分钟，两部分都是。资料分析用时40分钟，数量关系用时不太确定。当时刚开始逻辑部分的图形推理老是6个全错，然后就在论坛下载了图推经典100道题，大概花了一上午时间看完，看完之后对图推大概的做题思路就有了了解，正确率就上来了，一般是6个错1，2个。嗯，大家可以在行测资料下载区自己找几个，耐心看完，都会有用的。

有很多同学对于数量关系采用的态度是完全放弃，我反倒觉得这个部分应该是准备有立竿见影效果的部分。省考之前20天左右，我买了某图的数量分析模块宝典，过到一半的时候就要考试了，但是效果真是马上看得见。学了大概5天左右的时候我的数量分析部分用时25分钟，15个对13个，当时对这个正确率满意死了。个人认为，其中最有用当属化归为一法和倍数法。像牛吃草，十字相乘法这些方法虽然简单但是应用有限。我建议大家学了那些方法之后，尽量见到题就试着用，看一下到底能够应用于哪些题，刚开始的时候速度会慢，因为你要判断怎么用，用哪个。不过不要紧，一习惯就可以见题就来，到最后你都不知道你解题用技巧没。到了，这个阶段就会速度与正确率并存啦。

对于资料分析，当时就是死算，而且没有学过任何技巧，什么直除法压根没有用过，毕竟直接就上的真题，正确率还好，时间还行（言语和逻辑省下的时间全给了他了），所以在各种速算技巧上有不少缺失。当时就是每天一个小时专门用来做资料分析，提高自己的速度。做题的时候自己发现咦，这个题算完前两位就可以了，咦，这个题不用算的嘛，直接就能看出来······当时就是自己靠着这点小技巧一点一点提高速度的。

至于申论，哎，真是一言难尽啊，完全不知道该怎么学习。当时就是在论坛下了人民日报评论员文章，学着文章结构来安排自己的结构，偶尔能记几个优美句子，对于大作文真是快愁死我了，完全不知道该怎么写。不过，对于小题，哈哈，我还是会的。做题的时候先看题目要求，然后带着问题开始读题，读得时候将任何可能用到的信息都在草稿纸上罗列出来，这样读完一遍就把会用的信息都提炼出来了，这个过程40分钟足以，然后不是还要逻辑么，就把自己提炼出来的信息进行分类，相同的进行合并，在这提醒大家小题是按点采分的，要是不确定是否一样，那么就都写上，谁知道哪个词是关键词啊。这样做出来的题一般不会超字数，每次我都是特别短小精干的答案。还得加点料。嗯，我写的不够具体，建议大家到申论版看那些强人的帖子，还有具体范例。
然后，我就带着这些状态上了省考考场，120面试都没进。
